# Official Documentation Sources

Canonical Anthropic references for Claude Code customization development.

## Claude Code

- [Skills](https://docs.anthropic.com/en/docs/claude-code/skills) — SKILL.md structure, frontmatter fields, invocation control, supporting files
- [Sub-agents](https://docs.anthropic.com/en/docs/claude-code/sub-agents) — Agent definitions, delegation, preloaded skills
- [Hooks](https://docs.anthropic.com/en/docs/claude-code/hooks) — Lifecycle events, hook types, configuration
- [Memory](https://docs.anthropic.com/en/docs/claude-code/memory) — CLAUDE.md files, auto memory
- [Settings](https://docs.anthropic.com/en/docs/claude-code/settings) — Permissions, settings files, managed settings
- [Plugins](https://docs.anthropic.com/en/docs/claude-code/plugins) — Packaging and distributing skills
- [CLI reference](https://code.claude.com/docs/en/cli-reference) — Commands and flags

Also available at [code.claude.com/docs](https://code.claude.com/docs).
