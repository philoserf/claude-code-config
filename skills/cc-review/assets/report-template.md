# Report Template

Unified output format for cc-review reports. Three sections matching the three phases.

## Full Report

```markdown
# cc-review: {Name}

**Location:** `{path/to/target}`
**Type:** {Skill | Agent | Hook | Rule}
**Date:** {YYYY-MM-DD}

---

## Lint

| Check | Status |
|-------|--------|
| {check name} | {PASS / WARN / FAIL} |
| ... | ... |

**Findings:**
- {Specific finding with file:line reference}

---

## Score

| Dimension | Weight | Score | Weighted |
|-----------|--------|-------|----------|
| Effectiveness | 28% | {1-5} | {score × 0.28} |
| Clarity | 22% | {1-5} | {score × 0.22} |
| Best Practices | 17% | {1-5} | {score × 0.17} |
| Documentation | 15% | {1-5} | {score × 0.15} |
| Verification | 10% | {1-5} | {score × 0.10} |
| Trigger Coverage | 8% | {1-5} | {score × 0.08} |
| **Total** | | | **{sum}** |

**Tier:** {tier name} ({range})

**Per-dimension notes:**
- *{Dimension} ({score}):* {Evidence and rationale}

---

## Improve

| # | Finding | Dimension | Priority | Effort |
|---|---------|-----------|----------|--------|
| {N} | {finding} | {dimension} | {P1-P5} | {Low/Med/High} |

**{P1} — {Finding title}**
*{Dimension} · {Impact} impact · {Effort} effort*
Current: {what exists now}
Recommended: {specific change}
Why: {rationale}

{...remaining recommendations by priority...}

---
*Report generated by cc-review*
```

## Report Guidelines

1. **Evidence-based**: Cite line numbers, file paths, or quoted text for every score
2. **Actionable**: Improvements specific enough to implement without further research
3. **Proportional**: Dimensions scoring 5 get evidence only; 1-2 get detailed analysis
4. **P1 first**: Always list P1 recommendations before P2, P3, etc.

## Abbreviated Format

For batch comparisons across multiple customizations:

```text
{name} — {X.XX} {Tier} — {N} P1s, {N} P3s
```

## Comparison Format

When reviewing multiple customizations:

```markdown
| Name | Score | Tier | Strongest | Weakest |
|------|-------|------|-----------|---------|
| {name} | {X.XX} | {tier} | {dimension} | {dimension} |
```
