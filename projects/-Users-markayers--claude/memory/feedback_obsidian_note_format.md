---
name: Obsidian note formatting rules
description: No H1 headings, no horizontal rules, minimal frontmatter (created only), file name is the title
type: feedback
---

Generated Obsidian notes must not include H1 headings or `---` horizontal rules. File name serves as the title. Frontmatter should be minimal — only `created: YYYY-MM-DD` is required.

**Why:** User prefers clean, minimal notes. H1 duplicates the file name. Horizontal rules clash with frontmatter fences.

**How to apply:** Any skill that writes to Obsidian (session-review, etc.) should follow these rules. Start content with H2, never H1.
