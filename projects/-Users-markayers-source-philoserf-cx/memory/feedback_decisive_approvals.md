---
name: Decisive approvals
description: User approves with single letters/words — don't ask for elaboration
type: feedback
---

User approves designs and recommendations with terse responses ("a", "approved", "yes"). Don't restate or ask for confirmation.

**Why:** User values action over explanation. Terse responses are affirmative, not disengaged.

**How to apply:** Accept single-word approvals and move forward immediately. Don't ask "are you sure?" or restate what was agreed.
