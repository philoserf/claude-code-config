# reflection

Reflect on the current conversation to identify improvements for CLAUDE.md.

## Usage

`/reflection`

## Delegation

This command invokes the **improve-instructions** skill, which:

1. Analyzes conversation patterns (corrections, preferences, workflows)
2. Reviews current CLAUDE.md state
3. Proposes specific improvements
4. Implements approved changes

See the skill for detailed workflow and analysis patterns.
